## 1. CVP — каноническая формулировка

**Для** частных мастеров и малых услуг,
**которым** нужен канал онлайн-заявок, но некогда вести сайт,
**Самосайт** — это сайт, который ИИ собирает из источника о вашем деле (Яндекс.Карты,
Telegram-канал, 2ГИС, Avito, свой сайт, фото) и сам обновляет каждую неделю,
**в отличие от** конструкторов сайтов и SMM-агентств,
**наш продукт** не требует ни строчки текста от вас — только ссылку или фото того,
что у вас уже есть.

**Одной строкой (elevator):**
> Самосайт собирает вам сайт из того, что у вас уже есть, и сам приводит заявки —
> без вашего участия.

### 2.2 Hero — **три «сам», без eyebrow, без benefits stack**

```
# Сайт, который
# <mark>сам себя соберёт,</mark>     ← terracotta + underline highlight
# <mark>сам обновит</mark> и         ← terracotta no underline
# <mark>сам приведёт клиентов</mark> ← terracotta no underline

Покажите ссылку на ваше дело — карты, Telegram или визитку. Самосайт соберётся за
две минуты и дальше будет всё делать сам: обновлять цены, ловить заявки, отбирать
отзывы. Вам остаётся только работать с клиентами.

[ ссылка на соцсеть или Я.Карты ]  [ Собрать мой Самосайт → ]

🛡 Первый месяц — бесплатно. Самосайт сам напомнит, если решите продолжить. Карта не нужна.
```

**v2.1 правила H1:**
- Три «сам» — обыгрывают название (`Самосайт` = сам + сайт).
- Запятая после «соберёт» живёт **внутри** первого `<span>` чтобы не повисать в
  начале строки на узких viewport.
- Каждая «сам …» фраза — `whitespace-nowrap inline-block`, wraps как единый блок
  (не разрывается между «сам» и глаголом).
- **Без точки в конце** — H1 punch'нее без неё.
- Underline highlight только на первой фразе («сам себя соберёт») — она visual anchor.

Под формой — короткая микрокопия про supported-список **не нужна** (она теперь живёт
в секции Platforms). После того как юзер вставит URL — под полем появляется зелёная
плашка `✓ Распознали: <source> — <counts>` (см. §5 ниже).

Под Hero — **одна fallback-ссылка** «📷 Загрузить фото работ, скриншот профиля или
визитку». Ссылка «закрытый TG-канал» переехала в FAQ — это слишком узкий сценарий
для самого видного места.

### 3.5 Bot distinction (важно — двух ботов нельзя путать)

| Бот | Хэндл | Контекст |
|---|---|---|
| **Intake** | `@SamositeIntakeBot` | Юзер добавляет его **админом в свой TG-канал-источник** на 5 минут, чтобы мы могли парсить посты канала. Появляется в экране #4 «Канал приватный — пустите бота». См. ADR-0005 (Tier 1). |

## 5. Микрокопирайт — поле ввода на Hero

Универсальный паттерн: `✓ Распознали: <тип источника> — <counts>` + popover override.

API: `GET /api/preview?url=<encoded>` → `{ source, tier, counts, override_options }`.
Дебаунс 300ms на стороне фронта.

| Tier | UI |
|---|---|
| `ok` (Я.Карты, TG-канал, 2ГИС, Avito, свой сайт) | ✓ Распознали: `<source>` — `<counts>` |
| `soon` (VK, IG, WhatsApp-каталог, YouTube) | ⚠️ `<source>` скоро будет — оставьте email, напишем когда добавим [parallel CTA: или создайте из фото сейчас] |
| `unknown` (any other URL) | Не узнали источник. Какой это? [open input] — мы добавим |
| photo upload | ✓ Фото — N файлов загружено |
| не URL | Введите ссылку или загрузите фото → |

«Не то?» рядом с зелёной плашкой `ok` → popover со списком из `override_options`.

## 7. Антипаттерны — что НЕ говорим

| Чего избегаем | Почему |
|---|---|
| «ИИ-генератор сайтов» | Категория-средство, не результат |
| «Schema.org», «sitemap», «индексация» | Жаргон SEO, мастер не понимает |

| «Создаст сайт за 2 минуты» как главный hook | Продаём автономность, не скорость |
| «Пока вы работаете» | Буквальная трактовка: «а если я не работаю?» |
| Триал в h1/CVP | Оффер ≠ позиционирование |
| Перечисление 4+ фич ИИ в подзаголовке | Перегруз |
| Латинское `Samosite` в публичной копи | Бренд читается по-русски |
| Один TG-бот вместо двух | Путаница intake vs notifications. См. ADR-0011 |
| **Eyebrow перед H1** (v2) | Перегружает Hero, выбран более чистый дизайн |
| **Auto-detect для контакта** (v2) | Запутывает юзера, мешает radio-выбору |

## 8. Open questions

- **OQ1.** SLA авто-обновления по каждому типу источника. Для статичных (PDF/JPG)
